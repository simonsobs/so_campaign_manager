SO Campaign Manager Documentation
===================================

Welcome to the SO Campaign Manager documentation! This package provides tools for managing and executing mapmaking campaigns on high-performance computing resources, specifically designed for the Simons Observatory project.

Features
--------

* **Campaign Management**: Organize and execute multiple workflows with different policies
* **Resource Management**: Configure and manage computing resources (nodes, cores, memory)
* **Workflow Support**: Built-in support for ML mapmaking and null test workflows
* **SLURM Integration**: Automatic resource allocation and job submission
* **Flexible Configuration**: TOML-based configuration system
